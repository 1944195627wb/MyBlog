Clash for Linux

Linux 环境配置

很多小伙伴都知道Clash这个强大的东西，但是并不清楚Clash竟然如此强大，可以在Linux系统上部署并使用，今天我将这个方法告诉大家！

Linux分为桌面版和服务版两种，桌面版是有图形界面的，server版仅有命令行的。下面会以Ubuntu系统为例分别讲解这两种方案。

在此之前，你需要先购买一个VPN，获得一个订阅链接，这个订阅链接是后面必须的配置参数。

这里推荐两个途径，相对来说都比较便宜。

阿伟云

性价比机场

一.Ubuntu桌面版

1.下载资源

小猫咪Linux图形版

2.启动程序

将第一步中，“小猫咪Linux图形版”的内容复制到ubuntu图形版机器中，然后双击运行cfw，就可以看到熟悉的页面，将订阅链接填入Profile就可以使用了。

第一种方法可能失败，如果不行的话，在根目录文件打开终端运行以下命令行即可启动程序

./cfw

3.配置节点

后面配置的话跟Windows就差不多了，可以点击Start with Linux就可以开机自启动了

就是你在前面的网站中购买好节点好后，然后复制其中的订阅地址，按下回车或者点击Download即可，然后选择一个延迟比较低的节点即可。

4.浏览器配置

Linux用是火狐浏览器的话，需要配置一下网络端口。

首先打开Clash程序，首页选择Port

编辑端口，输入以规定一个端口号，这里我选择的是8080

编辑好之后打开火狐浏览器的设置，然后搜索端口，设置为手动代理，然后端口号也设置成8080，按以下配置即可

5.验证是否配置成功

打开浏览器，添加网址https://www.google.com
观察是否能够进入搜索页面，如果能够出现谷歌的搜索框，那么恭喜你配置成功！

二.Ubuntu服务版

1.下载资源

小猫咪Linux服务版

利用git clone将项目克隆至本地，或者直接将项目打包后的zip文件下载至本地

2.配置参数

将第一步中克隆的”小猫咪Linux服务器版”的内容复制到ubuntu server机器中，然后cd到该文件夹目录中

修改.env文件 将其中的 export CLASH_URL=”订阅链接”,以下修改方式习惯用哪个就用哪个

vim .env或者 nano .env或者gedit .env

3.开启服务

运行以下命令

sudo bash start.sh

start.sh是开启服务的脚本

4.配置环境变量

source /etc/profile.d/clash.sh

5.开启代理

proxy_on

6.验证是否配置成功

curl https://www.google.com/

若出现返回的一堆网页代码，即配置成功

7.关闭代理

proxy_off

注意：在使用过后记得要关闭代理！！！
